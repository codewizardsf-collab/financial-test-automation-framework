# Financial Test Automation Framework

QA project for financial trading and payment workflows. It models E2E scenarios, API assertions, contract checks, and performance-risk scenarios.

## Resume Fit

- Playwright/Cypress style E2E thinking.
- API contract validation.
- Financial calculation boundary tests.

## Current Scope

- Node test examples using built-in test runner.
- Gherkin-style scenario documentation.

## Production Next Steps

- Add Playwright browser tests.
- Add Pact provider/consumer contracts.
- Add Allure reporting.
